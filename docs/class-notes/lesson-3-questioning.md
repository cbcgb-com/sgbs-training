# 預備查經題目

作為小組查經帶領著，我們需要學會的一門技術和藝術就是如何通過問題來引導大家的思路。

## 提問題的重要性

為何要提問題呢？為何不以講道的方式進行小組查經呢？
要回答這個問題的話，我們需要理解為何「問題」如此的有力量。

當我們在一個「講道」的環境時，
台上的講員處於「積極」狀態。
他/她已經提前下功夫，與經文有自己的一番掙扎。
而台下的聽眾則比較高概率會處於「消極」狀態。
當然，不排除聽眾當中肯定有積極聽講道的聽眾。

反觀一個以「提問」為主的小組查經，
被提問的組員們不得不在被提問之後來一個回復。
因此，提問題相當於一個邀請，邀請組員們參與在討論之中，
與經文的內容掙扎一番，並且做出自己的回應。
如此的討論方式對於組員們來說更加積極，
也因此更能記得住那周的查經內容。

## 有方向的查經

與此同時，作為查經帶領者，我們需要給小組查經的討論定下一個方向。
沒有方向的查經如同聖經所描述的，瞎子領瞎子，盲目地為了查經而查經 --
寧願把那種查經改成禱告會好了，如此會更造就人。

假如還沒有開始做的話，鼓勵同工們在擬定查經計劃的時候，
邀請一兩位同工給大家定一個整體的方向。
比如說，查約翰福音的時候，不一定非得要每章每節都查完，
可以按照一個事先定好的主題（例如「榮耀」的主題）
挑選合適的經文來學習（例如「耶穌所行的神跡」），
然後鼓勵同工們在預備查經的時候，
根據大主題擬定當天的主題
（例如，開篇神跡加拿的婚宴中可以探討這個神跡顯出耶穌榮耀的哪方面）。
這份工作自然是要留給有經驗的同工，
但也要記得找一位有心學習的同工一起下手，不然的話少了傳承的機會。

在這個前提下，每周的查經可以更有方向感，
你作為預備查經的同工也可以更容易挑選合適的查經方向。
假如你也可以考慮到組員們的需要，
而不是僅憑個人的感動和偏向而已，
你的小組的查經成為一個你服事他人、造就他人的良好途徑。

## 提問題的原則

那麼，如何出查經題目呢？是不是還是一臉「懵」，感覺有點不知所措呢？
假如你有看過別人帶領查經的話，你可能會認為，有沒有什麼模版可以跟著走呢？

別擔心，在這個部分中，我們會嘗試給你一些可以考慮和嘗試的方向。
不過我們會強調一點：查經的每個元素都必須要有存在的目的。
與其提供一個可能會成為你的拐杖的「完整查經模版」，我們會提供給你一些可以考慮的方向。

這個section背後的一個假設就是：你已經做好前期預備了，
你也已經擬定了一個查經重點和方向，
而你現在掙扎的是要怎麼出題才不會導致小組查經無效。

### 要不要有破冰題？

答案是：「看你的處境！」

如果一個破冰題目可以引導組員們去思考最後你想要大家思考的方向的話，就不妨把它加進去吧！
又或者，假如你的破冰題目可以讓大家以一個放鬆的方式進入到查經的狀態里，也可以考慮把它加進去。

千萬不要為了有破冰題目加一個破冰題目！

### 題目要如何問？

當你在思考每一道題目要如何出的時候，我們的建議是按照「減少突兀情況」的原則來出題。
那麼，什麼樣的題目類型會導致「突兀情況」出現呢？請參考以下的出題反模式（anti-patterns）：

- 題目不闡明是否要思考還是要看經文找答案
- 題目一下子就問很大的概念
- 題目可以用簡單的一句的話就回答
- 題目有模糊不清、容易被誤解的地方
- 題目對於背景知識的要求高過組員們已有的裝備

如果要避免這類情況的話，可以在設計討論題目的時候考慮下面的一些問題：

基本考量：

- 題目是不是可以用一句「是」或「不是」來回答？
    - 此類問題有它的用處，但查經的時候不能全部都是。
- 題目是不是可以直接從經文裡頭找出答案？
    - 如同上述的問題類型，查經時不能全部都是。
- 題目有沒有含糊不清的地方，或者會不會容易被小組組員們誤會？
    - 這當中需要依靠你對小組組員們的認識。

深入思考：

- 題目之間有沒有連貫性或複雜度的層次，來輔助大家一步一步建立對經文的認識？
- 設身處地去想一想你對小組的認識，大家碰上這道題目會怎麼回答？
- 順著題目下來，每一道題目有沒有足夠的鋪墊輔助大家回答？
- 你所出的題目，是不是可以在沒有正式的帶領著的情況下，依然可以被小組使用呢？

### 題目要如何編排？

當你在思考如何編排題目的時候，
我們的建議是從簡單的題目開始下手，
一步一步深入到需要更多思考的題目。
一開始如果來一個很大的問題，
你會高概率發現小組無法討論下去，
氣氛也會變得一度尷尬，後面的討論也可能會比較吃力一些。

千萬要記得：每一個查經題目必須要有它存在的意義！不要為了濫竽充數而添加題目。

## 有什麼樣的工具可以輔助設計清楚易懂的討論題目呢？

### 題目中的提示

問題中的提示（prompts）可以輔助大家參與。假如你的用意時要大家：

- 從經文中找答案，不妨加一個**「根據經文...」**這樣的一句話？
- 單獨思考後再分享，不妨加一句**「請單獨思考一下再分享，...」**？
- 一起討論，不妨提示大家**「請一起思考並討論...」**？

提供輕微的架構可以使得大家更容易參與進來！

### 把「為什麼」類型的問題問得仔細一些

為什麼呢？🤣

主要原因是因為「為什麼」問題背後有很多未知的因素。

首先，一個沒有任何限定條件（qualifier）的問題會令組員們以為這道題目有個標準答案，
因此而絞盡腦汁想盡辦法把思路朝向標準答案走。
而如果你的問題本身是想邀請大家去思考一番的話，這樣會限制大家的思路。

其次，當我們問「為什麼某某某會這麼做」之類的問題的時候，
被問到問題的組員可能會想：「組長這是在問來龍去脈還是耶穌的意圖？」
帶領者的意圖不明，會導致組員們的思路不清。

如果要闡明你背後的意圖的話，不妨考慮把題目問得仔細一點。

再次借用約翰福音迦拿婚宴為例，
與其問：「為什麼瑪麗亞跟耶穌說『酒沒了』呢？」，
不如問：「瑪麗亞跟耶穌說『酒沒了』背後可能在想什麼呢？」
一方面後者出題者思路比較清楚（要大家思考，不要去經文找答案），
另一方面也比較有代入感，邀請大家換位思考，從人物的角色去體會他/她的觀點和感受。

## 評估以下每一組聖經學習問題集

以下是兩組針對約翰福音2:1-11段落的聖經學習問題集。請以觀察員的角色對它們進行評論。

???+ 問題集1

    破冰題目： 你有因为身边的人做了某件特别的事情而印象很深刻，特别佩服TA的经历吗？对你有什么影响？

    討論題目

    - 问题一：这一段经文记载了什么事情？
    - 问题二：1 ３ 5 观察 耶稣母亲在宴席中扮演什么角色？
    - 问题三：３４耶稣的话想要表达什么？
    - 问题四：5 玛利亚是如何回应耶稣的回复的？
    - 问题五：6 观察题， 洁净的规矩， 两三桶水？
    - 问题六：7 8 用人是怎么做的？9 他们为什么要这么做？他们可能是怎么想的？
    - 问题七：10 11 神迹指什么，究竟神在哪？ 为什么门徒会信他？
    - 问题八：（对慕道友）如果你就是在场的门徒，你也会信吗？在今天的你读了这段经文会愿意相信耶稣吗？
    - 问题九：耶稣为什么要这样做？如果你是耶稣的话你会做一模一样的事情吗? 为什么不是直接在酒杯里变出酒来？

    進一步思考題目

    - 思考題一: 耶稣对玛利亚吩咐的回应对你有什么启示？
    - 思考題二: 玛利亚对耶稣回答的回应，以及用人对耶稣吩咐的回应对你有什么启示？

    總結點

    - 第一點: 只有完全顺服于神的人，才能经历并见证主奇妙的作为
    - 第二點: 不应该要求主立刻应允我们的祈求，因为祂有自己最美善的时间表。

???+ 問題集2

    破冰題目：在你生活中，有沒有經歷過某件普通的事情卻帶來意想不到的驚喜？那次經歷對你有怎樣的影響？

    討論題目：

    - 這個故事發生在哪裡？發生了什麼事件？
    - 經文中提到的主要人物是誰？
    - 婚宴中遇到了什麼問題？
    - 當耶穌的母親向他說話時，耶穌是如何回應的？

    進一步思考題目

    - 你覺得為什麼耶穌在第4節中以這樣的方式回應他的母親？他可能在暗示什麼關於他的使命或時間？
    - 潔淨用的水缸有什麼重要性？為什麼耶穌選擇這些特定的水缸來行神蹟？這可能說明他如何看待宗教傳統和儀式？水變為酒的神蹟是否象徵著更深層次的屬靈轉變？
    - 經文提到管宴席的並不知道酒從哪裡來，但舀水的僕人知道。這可能暗示了什麼關於神的工作方式？
    - 第11節提到耶穌的神蹟是“顯出他的榮耀來”。這說明了神蹟在耶穌事工中的作用是什麼？

    - 你如何看待瑪利亞在這個故事中的角色？你曾有過類似把需求或擔憂帶到神面前的經歷嗎？那次經歷如何？
    - 耶穌將水變為酒，把一個潛在的尷尬變成了美好的事物。你是否覺得在你生活中有些地方神也有能力帶來轉變？這可能是什麼樣子？
    - 瑪利亞對僕人說“他告訴你們什麼，你們就做什麼”，強調了順服的意義。今天，順服耶穌在你生活中是怎樣的表現？
    - 迦拿的神蹟展示了神的慷慨（在最後提供最好的酒）。這如何反映你對神在你生命中供應的看法？

    總結點

    這個故事講述了轉變和耶穌所提供的豐盛恩典。將水變為酒的第一個神蹟象徵著耶穌如何帶來新生命、喜樂以及在平凡時刻中的意外祝福。儘管他的“時刻尚未到”，這個小而個人的行動預示著他通過死而復活所成就的更大工作，不僅僅是將水變為酒，而是將我們從破碎轉變為完全。

## 如何思考在查經中使用GenAI？

我們認為生成式人工智慧（GenAI）是一個強大的工具，可以輔助我們進行查經，但不應該用來取代我們的思考和反思。

查經的主要目的是增進我們的信心，而個人與聖經的深入對話是這個成長過程中不可或缺的一部分。

如果我們僅僅依賴GenAI來生成討論問題，而不進行個人的思考和反思，我們就可能錯失與上帝話語真正對話的寶貴機會。

因此，我們應該將GenAI視為一個輔助工具，它可以啟發我們的思考，擴展我們的視角，但絕不能替代我們自身與聖經的互動和領悟。

在這個基礎上，我們相信GenAI可以成為一個有價值的資源，然而，需要謹慎使用！

??? info "GenAI使用的正面例子"

    以下是一些具體任務，我們認為GenAI可以協助我們，而不會削弱我們的個人反思、與上帝話語的對話，以及與其他信徒的交流。

    1. 將你的問題從母語翻譯成小組的主要使用語言。
    2. 翻譯非母語的聖經學習資源。
    3. 生成一個忠實但不一定深入的總結，以記錄查經討論內容。（忠實指準確性和真實性，深入則指在上帝話語指引下，從討論中進行個人反思和獲得更深層次的洞察。）
    4. 利用生成的文本作為提示，提醒自己可能遺漏的話題，無需討論。
    5. 將GenAI作為「第二讀者」，檢查你的問題和答案中可能存在的基本錯誤，但不能依賴它來理解經文。
    6. 作為搜索關鍵詞生成器，協助你使用搜索引擎深入研究某個主題。
    7. 作為創意來源，生成一些破冰問題。
    8. 生成輔助查經的討論活動。

??? info "GenAI使用的負面例子"

    以下是一些我們認為最好避免使用GenAI的任務：

    1. 將GenAI作為神學討論的主要對話對象。
    2. 完全依賴GenAI生成查經問題，而不進行個人思考。
    3. 將GenAI視為歷史事實的權威來源，而不進行交叉驗證。
    4. 將GenAI產生的內容視為神學洞察的唯一來源，忽視傳統神學研究和個人靈修。

??? info "使用GenAI的一般原則"

    1. 使用GenAI時要保持透明，明確說明使用情況。
    2. 將GenAI作為輔助工具，增強而非取代自己的思考能力。
    3. 對GenAI生成的內容進行仔細核實，確保每個細節的準確性。
    4. 對於使用GenAI生成的內容，要能夠解釋並為每一部分負責。
    5. 不要為了達到目的而濫用GenAI，應該遵循道德和倫理準則。
